# Arma Reforger Enforce Script - Coding Rules

## Language Rules

- File extension is `.c` but language is Enforce Script (NOT C/C++)
- All variables must have explicit types (statically typed)
- Default access is public (no keyword needed)
- String formatting uses `%1`, `%2` tokens (1-indexed), NOT `%s`/`%d`
- Vector literals are strings: `vector v = "1 0 1";` NOT `vector v = {1, 0, 1};`
- Array literal shorthand: `array<int> a = {};` creates empty array
- `foreach` works on arrays: `foreach (int item : myArray) { }`

## Naming Conventions

- Member variables: `m_` prefix with Hungarian type notation (`m_bBool`, `m_iInt`, `m_fFloat`, `m_sString`, `m_vVector`, `m_aArray`, `m_mMap`, `m_eEnum`)
- BI game classes: `SCR_` prefix
- Mod classes: Use your own prefix (e.g., `STAVKA_`, `ACE_`, `MY_`)
- Enums: `E` prefix (e.g., `EGameState`, `EDamageType`)
- RPC naming: `RpcDo_` for server-to-client, `RpcAsk_` for client-to-server

## Null Safety

- ALWAYS null-check after `Cast()` - returns null on failure, crashes on dereference
- ALWAYS null-check after `FindComponent()` - may return null
- ALWAYS null-check chained calls: `GetGame().GetPlayerController().GetControlledEntity()` can fail at any step
- Use early returns: `if (!comp) return;`

```enforce
// CORRECT
SCR_AIGroup group = SCR_AIGroup.Cast(entity);
if (!group) return;

// WRONG - will crash if cast fails
SCR_AIGroup group = SCR_AIGroup.Cast(entity);
group.GetAgentsCount();  // null dereference crash
```

## Override Rules

- ALWAYS call `super.MethodName()` in overrides unless intentionally replacing behavior
- In `modded` classes, skipping `super` breaks other mods in the chain
- Call `super` at the beginning unless you need to run code before parent

## Memory Rules

- Use `ref` for class member variables that own objects
- Use `autoptr` for function-local temporaries only
- NEVER store `autoptr` in class members (object dies when scope exits)
- Keep `ref` to callbacks alive (RestCallback, ScriptInvoker handlers)

## Performance Rules

- AVOID `EOnFrame` unless truly needed - prefer event-driven patterns
- Use `GetGame().GetCallqueue().CallLater()` for periodic/delayed work
- Guard expensive logic: `if (!Replication.IsServer()) return;`
- Use `SetEventMask(owner, EntityEvent.FRAME)` only when frame updates are required
- Clear event mask when done: `ClearEventMask(owner, EntityEvent.FRAME)`

## Replication Rules

- Game logic runs on SERVER only
- Visual/audio effects can run on any machine
- `[RplProp]` can only be declared in original class, NOT in `modded` classes
- Only server should write to `[RplProp]` properties
- Call `Replication.BumpMe()` after mutating complex RplProp types
- Use `[RplProp]` for persistent state (JIP-safe), NOT one-shot RPCs
- Use `RplChannel.Reliable` for critical state, `Unreliable` for frequent updates

## AI Rules

- AI groups need 1-2 seconds to initialize after spawning before accepting waypoints
- Use `GetGame().GetCallqueue().CallLater()` to delay waypoint assignment
- Check `group.IsInitializing()` before assigning waypoints
- Set `wp.SetCompletionRadius()` explicitly on waypoints

## Mod Structure Rules

- Scripts go in `Scripts/Game/` directory
- Modded class files go in `Scripts/Game/GameCore/Modded/` by convention
- Every ScriptComponent needs a companion `*Class : ScriptComponentClass`
- Resource references use `{GUID}path` format
- Base game dependency GUID is always `"58D0FB3206B6F859"`
- Never edit `resourceDatabase.rdb` manually - auto-generated by Workbench

## Common Patterns

### Delayed Timer in GameMode

```enforce
modded class SCR_BaseGameMode {
  protected float m_fTimer = 5.0;
  protected bool m_bRan = false;

  override void EOnFrame(IEntity owner, float timeSlice) {
    super.EOnFrame(owner, timeSlice);
    if (m_bRan) return;
    m_fTimer -= timeSlice;
    if (m_fTimer <= 0) {
      m_bRan = true;
      DoMyThing();
    }
  }
}
```

### Surface Height Query

```enforce
float y = GetGame().GetWorld().GetSurfaceY(x, z);
vector pos = Vector(x, y + 1, z);  // +1 to avoid clipping into ground
```

### Spawn Entity at Position

```enforce
EntitySpawnParams params = new EntitySpawnParams();
params.TransformMode = ETransformMode.WORLD;
params.Transform[3] = position;
Resource res = Resource.Load("{GUID}path.et");
if (!res || !res.IsValid()) return;
IEntity ent = GetGame().SpawnEntityPrefab(res, GetGame().GetWorld(), params);
```
